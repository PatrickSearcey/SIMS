How do I highlight or unhighlight a site as "real-time" (grey) site or "non-realtime" site, and vice versa?

The site must be first be designated as having telemetry (or not) in GWSI. Change the telemetry flag in Instruments (field 28, columns 3,4, or 5) to "Y" as appropriate. Column 3 is for land telemetry, column 4 is for radio telemetry, and column 5 is for satellite telemetry. A "Y" in any of these columns will designate the site as real time.

If you want also want the site to show on real-time web pages, set the web-ready flag (field 32) to "Y".

If you want to remove the site from the real-time list, change the flags to "N".

Run GWCHECK to push any changes to NWIS. The site will appear on the appropriate list in SIMS in 75 minutes or less.

How do we handle discontinued sites in SIMS?

Once a site is "registered" in SIMS, it should stay there. Should you recommission the site in the future, the documentation will then be readily available in electronic format.

Move the sites to the inactive list if they aren't there already by setting the appropriate Agency Use Code. See How do I highlight or unhighlight a site as "real-time" (grey) site or "non-realtime" site, and vice versa?

After changes are made in the NWIS sitefile, GWCHECK should be run to push the changes to NWIS.
